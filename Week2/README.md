## Introduction to Node.js  
( a runtime environment that allows JavaScript to run outside the browser.)<br>
Node.js plays a crucial role in modern web development and is widely used for building servers, APIs, and AI-powered applications.<br> Understanding the basics of Node.js will help you prepare for the AI integrations coming up in Week 3.

### Quick Setup

1. Install Node.js on your system.
2. Verify the installation:

```bash
node -v
npm -v
```

3. Create a file named `app.js`:
```javascript
console.log("Hello from Node.js!");
```

4. Run the file:
```bash
node app.js
```
