# Finance Tracker

A multi-page Finance Tracker built using HTML, CSS and JavaScript.

## Features
- Add Income & Expenses
- Dashboard
- Transaction History
- Spending Reports
- AI Financial Insights (Gemini)
- AI Finance Assistant
- Settings (Budget, Dark Mode, Reset)

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- Google Gemini API
- Chart.js

## Run

npm install
npm start

Open:
http://localhost:3000
